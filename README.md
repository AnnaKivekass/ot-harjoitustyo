# Ohjelmistotekniikka, harjoitustyö

Teen harjoitustyönä **sovelluksen juoksutreenien seurantaan**, jonka avulla voi *tallentaa ja analysoida harjoituksia*.

---

## Sovelluksessa on käytössä invoke, jonka avulla voi suorittaa yleisimmät komennot projektin juuressa

- Sovelluksen käynnistys komennolla 

poetry run invoke start

- Sovelluksen testien ajaminen komennolla

poetry run invoke test

- Sovelluksen testikattavuusraportin saa generoitua komennolla

poetry run invoke coverage-report


- Sovelluksen pylint raportin saa komennolla

poetry run invoke lint
